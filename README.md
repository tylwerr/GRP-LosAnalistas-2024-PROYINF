## Requerimientos para el programa

Para poder correr el programa y mostrar el prototipo/avance, debe tener instalado lo siguiente:

Lenguaje:
- Python 

Librerías del lenguaje:
- pydicom 2.4.4
- numpy 1.26.4
- pillow 10.3.0
- tkinter 8.6

Luego debe descargar las carpetas presentes: DATOS_DICOM , Visualizador.

Una vez descargado, ejecute el archivo "visualizador.py" para que comience el programa, para salir solo cierre la ventana.
